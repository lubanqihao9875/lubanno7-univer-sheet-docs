介绍 ​

Lubanno7 Univer Sheet 是一个基于 Univer 构建的高性能 Vue 2 电子表格组件。它提供了丰富的功能和高度的可配置性，适用于各种复杂的表格需求。

特性 ​

  • 🚀 高性能渲染: 基于 Univer 引擎，支持大数据量表格的流畅渲染
  • 📊 丰富的功能: 支持嵌套表头、单元格编辑、数据验证、权限控制等
  • 🎨 高度可定制: 提供丰富的样式配置选项，满足不同的视觉需求
  • 🔧 灵活配置: 支持多种单元格编辑器，包括下拉选择、只读模式等
  • 🔒 权限控制: 内置完善的权限系统，支持细粒度的权限管理
  • 📋 丰富事件: 提供完整的事件系统，便于与业务逻辑集成

适用场景 ​

  • 数据管理系统
  • 财务报表系统
  • 项目管理工具
  • 数据分析平台
  • 配置管理界面
  • 任何需要复杂表格功能的应用

浏览器支持 ​

  • Chrome ≥ 88
  • Firefox ≥ 78
  • Safari ≥ 14
  • Edge ≥ 88

版本要求 ​

  • Node.js ≥ 14
  • Vue.js ≥ 2.5.17
